nginx 的编译安装

需要 pcre,zlib,openssl 库支持

依赖库 pcre

  1. 去pcre 下载，注意nginx不能支持pcre2，所以选择形如 pcre-8.42.zip 的版本
  2. 解压、编译、安装(默认已经有编译所必需的build-essential)
    unzip pcre-8.42.zip
    cd pcrre-8.42
    ./configure
    make&&make install

依赖库 zlib

  1. 去zlib 下载
  2. 解压、编译、安装

依赖库 openssl

  1. 去openssl 下载
  2. 解压、编译、安装
